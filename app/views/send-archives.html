Submeter um arquivo:

Aqui você tem a possibilidade de submeter seus arquivos de estudo/resumo de forma a auxiliar seus companheiros a compreenderem os assuntos em questão, formulando assim um ambiente de estudos que tem como base o companheirismo.

1º passo: O que eu preciso saber?

Prometo ser breve! Antes de sair por aí colocando qualquer tipo de arquivo dentro do sistema, é necessário começar citando algumas regras que nós possuímos dentro do sistema. Para entender melhor essas regras, basta clicar aqui e visualizá-las.

2º passo: Quem poderá visualizar isso?

Conforme explicado anteriormente, nosso sistema tem como base a colaboração e companheirismo entre seus usuários, ou seja, todos aqueles que cursam as mesmas matérias e ou curso que você, poderão visualizar estes arquivos para com os quais você submeteu. E é exatamente por este motivo que requisitamos que você sempre confirme o tipo de arquivo que está "subindo no sistema".

3º passo: Como realizar este procedimento?

Clique ao lado para selecionar um arquivo Página anterior
Ainda não possui uma conta?
Criar uma conta em nossa aplicação é extremamente simples. Não acredita? Então basta clicar aqui que eu te mostro como é rápido e simples este procedimento.
Quais os tipos de arquivos permitidos?
As extensões de arquivos permitidas para se realizar esse upload são: pdf, docx(Word) e pptx(Power Point). Além disso, o tamanho máximo permitido para um arquivo de upload, é de 4MB.
